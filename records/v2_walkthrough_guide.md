# Walkthrough - Remote Desktop Visual AI Agent Built (v2)

**版本号**：v2
**记录日期**：2026-04-19

项目已成功在 `d:\Dev\autoplay` 中构建完成，严格遵循 PyQt6 透明图层设计与混合 AI 决策逻辑。

## 关键成果亮点

### 1. PyQt6 透明交互层
系统不再依赖 Web，而是使用原生 PyQt6 构建了一个真正的 OS 级叠加层：
- **置顶与穿透**：透明层始终位于远程桌面之上，且支持鼠标穿透（AR 模式下不干扰操作）。
- **左侧面板**：任务队列管理。
- **右侧面板**：资源占用监控（CPU/GPU/MEM）与实时执行日志。
- **中间审批窗**：展示 AI 生成的工作流，支持人工通过/驳回。

### 2. 混合 AI 决策引擎
- **本地优先**：尝试使用本地 YOLOv8n 和 Llama 3。
- **云端热切换**：当本地推理延迟过高或识别失败时，自动通过 `ModelManager` 降级到云端大语言模型。

### 3. 底层 AHK 执行器
- **物理模拟**：模拟真实点击与输入。
- **页面清空**：专门的 `clear_page()` 逻辑，确保任务循环间的页面“整洁”。

## 目录结构说明
- `src/overlay/`: UI 渲染与透明窗口逻辑。
- `src/ai/`: 决策引擎与模式切换逻辑。
- `src/execution/`: AHK 与任务状态机。
- `src/utils/`: 视觉采集与窗口对齐工具。
- `records/`: 项目版本化执行记录与指南。

## 快速运行指南

1. **配置检查**：确认 `config/default_config.json` 中的 `ahk_path` 指向正确的系统路径。
2. **目标准备**：打开向日葵或 Google 远程桌面。
3. **启动程序**：
   ```powershell
   .venv\Scripts\python main.py
   ```
4. **验证效果**：控制台应自动对齐到远程窗口，右侧日志面板开始输出同步信息。
