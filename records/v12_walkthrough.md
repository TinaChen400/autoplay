# Walkthrough V12: 万能窗口持锁与一键启动

本版本解决了 Agent 在多窗口、多标签页环境下“认生”的问题，并提供了极简的离线启动方式。

## 核心能力提升

1.  **HWND 句柄级死锁 (Space-Lock)**：
    - 引入了通过鼠标物理位置反查 HWND 的逻辑。
    - 用户按下空格后，Agent 将越过“标题匹配”，直接对底层窗口句柄进行 500ms/次的物理对位追踪。
2.  **批处理启动脚本 (Start_HUD.bat)**：
    - 封装了 `.venv` 环境与启动参数，为非技术性重启提供了 0 门槛方案。
3.  **状态持久化**：
    - 修复了 V11 切换窗口导致的吸附状态丢失，支持跨窗口、跨应用的瞬时切换。

## 使用指引

- **重启**: 双击 [Start_HUD.bat](file:///d:/Dev/autoplay/Start_HUD.bat)。
- **强制吸附**: 拖动面板覆盖目标窗口 -> 按下 **`空格`**。
- **退出**: 按下 **`Esc`**。
