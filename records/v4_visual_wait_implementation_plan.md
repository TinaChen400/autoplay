# Implementation Plan V4: 智能视觉增量检测与大图交互

本计划旨在为 `MSISkills` 增加智能视觉判定逻辑，确保在远程环境大图加载完成后，精准执行键盘交互。

## 核心技术选型 (OpenCV)

1.  **判定算法：** 使用 `cv2.absdiff` (绝对差值) 进行帧间的像素级对比，有效过滤静态背景干扰。
2.  **鲁棒性逻辑：** 
    *   **超时机制**：10 秒内未检出显著变化则报错，防止脚本卡死。
    *   **动态 ROI**：重点监测屏幕中心 70% 的区域，避免边缘通知或状态栏闪烁导致的误判。
    *   **稳定判定**：检测到变化后，需连续 2 帧保持稳定（差值极小）才判定为“图片已完全加载”。
