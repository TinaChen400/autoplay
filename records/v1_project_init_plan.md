# 远程桌面视觉 AI Agent 项目执行记录 - v1

**执行日期**：2026-04-19
**版本号**：v1
**状态**：已立项 / 规划中

## 核心任务目标
基于 PyQt6 透明图层技术，构建一个摆脱 Web 依赖、具备视觉增强（AR）能力、支持本地与云端模型协作的 OS 级 AI Agent 自动化系统。

## 已确认的技术方案
1. **纯本地化环境**：所有依赖、虚拟环境、缓存、模型文件均保存在 `d:\Dev\autoplay`，严禁写入 C 盘。
2. **核心架构**：
    - **PyQt6/PySide6**：负责透明穿透窗口与 AR 视觉增强绘制。
    - **双模型联动**：本地 Llama 3 8B (4-bit/8-bit 量化) + YOLOv8n + 豆包云端模型补位。
    - **任务控制**：AHK/mss 处理窗口句柄与底层执行，配合任务状态机。
3. **功能闭环**：任务配置 -> 1:1 坐标对齐画面同步 -> 工作流审批 -> 自动执行 -> 实时监测调试 -> 页面清空。

## 目录结构规划
```text
autoplay/
├── records/             # 项目执行记录 (当前文件夹)
├── .venv/               # 本地虚拟环境
├── models/              # 本地模型文件存储
├── config/              # 任务与识别规则配置
├── src/
│   ├── overlay/         # PyQt6 透明窗口与绘制逻辑
│   ├── ai/              # 模型调用接口与自动切换
│   ├── execution/       # AHK 执行与任务调度
│   ├── monitor/         # 调试、日志与监测模块
│   └── utils/           # 采集、窗口管理工具
├── logs/                # 运行日志输出
└── main.py              # 程序入口
```
