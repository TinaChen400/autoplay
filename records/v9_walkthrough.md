# Walkthrough V9: 150% 缩放下的 DPI 物理像素全链路对齐

本版本彻底解决了高 DPI 缩放环境下 UI 对位偏离、边框虚大以及用户无法手动干预的痛点。

## 核心技术突破

1.  **Win32 物理接口替换**：
    - 在 `window_lock.py` 中引入 `win32gui.GetWindowRect`。
    - 强制所有相关进程开启 `ProcessDpiAwareness(1)`。
2.  **拖拽保护逻辑 (Manual Override)**：
    - 引进状态机，检测到鼠标拖拽后自动解锁吸附，提供 [MANUAL] 模式。
    - 用户可通过 UI 按钮一键恢复 [AUTO-SNAP] 吸附。
3.  **几何对位修正**：
    - 精确计算了 Windows 窗口的隐形边框补偿（7 像素），实现了边框级别的像素级契合。

## 操作指南

- **自动吸附**: 启动后自动寻找 `MSI` 相关窗口进行 150% 物理像素对位。
- **手动调节**: 直接拖拽右侧控制面板，左下角变为 `[MANUAL]`，吸附暂停。
- **恢复对位**: 点击面板右侧出现的红色 `RESUME AUTO-SNAP` 按钮。
- **一键退出**: 点击 `QUIT` 或按下 `Esc` 键。
