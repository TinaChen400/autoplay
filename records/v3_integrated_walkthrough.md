# Walkthrough - 系统大集成与快捷键驱动 (v3)

**版本号**：v3
**记录日期**：2026-04-19

本项目已完成从视觉采集到 AI 决策、人工审批、物理执行的全链路集成，特别适配了 RTX 4080 高性能本地环境。

## 核心功能集成说明

### 1. Ollama AI 决策引擎 (Ready)
不再依赖不稳定的本地 DLL 编译。系统直接通过 REST API 对接您本机的 Ollama 服务，秒级响应 `llama3.1:8b` 的决策指令。

### 2. 全局快捷键控制 (Integrated)
基于 `pynput` 构建了非阻塞、全局生效的控制逻辑。您可以在远程窗口聚焦时直接通过键盘指令接管 Agent：
- **审批**：`Ctrl + Enter`
- **调试**：`Ctrl + D`
- **切换引擎**：`Ctrl + M`
- **紧急中止**：`ESC`

### 3. 透明图层交互中心
- **左侧**：任务点击 -> 触发 AI 异步分析 -> 自动生成工作流。
- **中间**：自动弹出绿色边框的审批窗，展示大模型逻辑。
- **右侧**：实时记录 AI 推测、执行动作与系统状态。

## 目录结构 (最新)
- `main.py`: 系统核心控制器
- `src/utils/shortcuts.py`: 键盘钩子监听器
- `src/ai/local_engine.py`: Ollama API 封装
- `src/execution/task_machine.py`: 逻辑步进状态机

## 测试步骤建议

1. **环境确认**：确保 Ollama 在后台运行，且已拉取 `llama3.1:8b` 模型。
2. **窗口对齐**：启动 `main.py`，透明层应自动吸附至远程桌面窗口。
3. **闭环测试**：
   - 点击左侧任务 -> 观察右侧 Log 显示“AI 分析中”。
   - 观察中间弹窗流。
   - 按下 `Ctrl + Enter` -> 观察 AHK 是否模拟了点击并最终执行了“页面清空”。
